Intro

This is alpha software and assumes that the current group of users will be tolerant of rough edges in all aspects of this product. There are a number of limitations in this software that need to be addressed. As this is an open source project you may be the answer.

Subsect uses the WebRTC protocol and the hosted sites can only be accessed by a browser that supports WebRTC data channel. This software has been tested with Chrome, Safari, Firefox and Opera. Microsoft's Edge browser is not currently supported.

Administration

Upon installing the app the Administration page opens. Host Name is the unique server name which will be your subdomain, yourhostname.subsect.net. This will become the host part of your sites URLs'. It is 4 - 20 lower case alphanumeric characters long. A password is required which is 4 - 20 upper/lower case alphanumeric (plus _ ) characters long. Email is optional and is currently not used but is planned to be for contacting hosts if needed.

Once the host name is submitted and accepted it can not currently be updated. The password and email can be changed. This can only be done through the app and on the device the server is running on. You do not need to know the current password to change it.

On exiting Administration to the Subsect Server main page the server should start within a few seconds. The server is running when the Start/Stop button is labled Stop and the start date time is to he right of it. If it is not running close the app out with the task switcher and relaunch it. Tapping Stop will stop the server sites on the device will not be reachable. Tap Start to restart the server. Always stop the server before exiting the app so users will be alerted the erver is not available.

The log button to the right shows a console screen which dumps some debug currently.

Bazaar

With the server running it is now time to load a site. Subsect has the equivalent of an "App/Play Store" which is called the Bazaar. It is early days in thee Bazaar with minimal functionality. After selecting the Bazaar option from the menu a list of available sites/services should appear. It is recommend that the TestSite be installed first. Tap the icon and accept install. This should only take a few seconds. After install completes return to main page and select Installed from Menu. TestSite should be listed.

If all is good, open an WebRTC compliant browser and enter the url yourhostname.subsect.net . A page with the tile Menu should open and TestSite should show. Tap/click item and it should open the site installed on your device. Try to add/edit a name entry. TestSite is largely just to confirm your server/database is operational.

Tools

Before using any tools STOP the server.

The Tools menu option supports four functions. EXPORT allows an individual site to be packaged for installation on another device or for backup. No data is exported only site logic and resources. Select from the list of current packages then tap EXPORT. IMPORT is the opposite. This is largely to support a side loading type function.

!! After using EXPORT or BACKUP DB you must close Subsect with the task manager and relaunch before doing any other operations !!

You can download/upload the export using the the appropriate option in the Menu page when you access yourhostname.subsect.net in the browser. You will need to login in Menu (in browser) to get these options. The password is the one setup in Administration.

BACKUP DB and RESTORE DB allow for the backup and export of datafiles. The BACKUP DB function will coppy the DB of each installed site and make a tar of all into one file. Again download to the browser using the Menu page. When uploading and downloading your browsing device needs to be on the same network (WiFi router for example) as your Android device. You can use the browser on your device to download and then copy that off as this is always on the same network.

Settings

The options here are for using an alternative connection server. If developing a site it is more convenient to use a local server. The setup of a development environment is beyond the scope of this help. Just keep the "Connection Server : subsect.net" option checked.

Installed

To remove an installed site, go to the Installed menu option and tap the "Remove" button next to the icon the site to be removed. All data associated with this site will also be removed. After removing a package it is recommended that the server be closed and the Subsect app closed with the task manager and then restarted.
